This is a tool that I am making to aid me managing the tracks/car configs for my Assetto Corsa server and in the future allow me to move files to my server/update config for my servers website when I add new cars/car categories to the site.

The goal from this was to learn WinUI and improve on the broken WinForm version I created previously. There's a lot of manual copy/pasting when adding new modded cars to the server as well as keeping up with car categories for the server site.
Content Manager already allows editing the entry list/track config but my server is hosted through a self hosted AMP Game Control Panel server and not Content Manager so I still have to copy/paste. This is a project to learn some new things and create something I can use to manage my own server
